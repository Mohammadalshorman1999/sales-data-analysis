# Sales Data Analysis using Python

## Project Overview

This project analyzes a real-world sales dataset to extract meaningful business insights.
The workflow includes data cleaning, feature engineering, exploratory data analysis, and visualization.

## Tools Used

* Python
* Pandas
* Matplotlib
* Jupyter Notebook / Google Colab

## Data Cleaning Steps

* Converted `ORDERDATE` to datetime format
* Checked for duplicate records
* Verified logical consistency of quantity and sales values
* Created time-based features (year, month, day)

## Analysis Performed

* Total sales by year
* Sales by product line
* Sales by country
* Sales by month
* Top customers by revenue
* Sales by deal size
* Sales trend over time

## Key Insights

* Highest sales were recorded in **2004**.
* **Classic Cars** was the top-performing product line.
* The **USA** generated the highest revenue.
* **November** was the strongest sales month.
* A small number of customers contributed a large share of total revenue.
